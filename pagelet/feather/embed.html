内容嵌入是feather从fis中继承过来的3种能力之一

在html和css中的用法一致，比如:

<!--该图片的内容会以base64的方式引入进img标签中-->
<img src="/static/1.png?__inline" />

<style src="/static/1.css?__inline"></style>
<!--
<style type="text/css">
*{
	font-size: 12px;
	color: #ccc;
}

div{
	border: 1px solid #000;
}
</style>
-->

/*将/static/1.css的内容嵌入至当前css文件中*/

@import url('/static/1.css?__inline');

在js中使用：

var div = document.createElement('div');
div.innerHTML = __inline('/static/1.txt');	//将1.txt的内容赋给div的innerhtml属性
document.body.appendChild(div);
