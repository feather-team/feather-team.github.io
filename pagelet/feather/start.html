安装feather：

  • 安装nodejs，因feather为npm包，所以需要安装nodejs环境。
  • 安装java，并配置环境变量，feather中的webserver依赖于java。
  • 安装php(只需要安装一个解压版，将php添加至环境变量即可, linux和mac平台可用源码包安装php，只需要安装最简单版本即可)
npm install -g feather

feather的4条命令，分别为：server, release, install, init

  • server：feather server的一些操作
  • release：编译产出操作
  • install：安装feather插件
  • init: 命令行创建项目
feather server start --timeout 30 #timeout可在启动服务器超时使用，默认超时时间为15s

feather有个比较简单的目录结构规范：

├── component
├── page
│   └── index.html
├── static
├── test
├── feather_conf.js
├── feather_rewrite.php
├── feather_compatible.php
  • component：模板小组件目录
  • page：page级别页面存放目录
  • static：page级别页面静态资源和其他静态资源存放目录
  • test：测试文件目录
  • feather_conf.js: feather的配置文件, 此文件为必须存在的文件，点击查看常用配置
  • feather_rewrite.php: url转发文件，php项目中使用到转发功能的可使用此文件。
  • feather_compatible.php：兼容php调用中的错误或者执行一些php代码，比如某一个在模版中使用的php函数，本地无法模拟，可将其简单的定义在这个文件中。

也可通过init项目脚手架来创建项目：

feather init -n web -m common -c utf-8 feather-project-common
